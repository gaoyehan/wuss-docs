# QR Code 二维码

### 使用指南

在 page.json 中引入组件

```json
"usingComponents": {
  "w-qr-code": "wuss-weapp/w-qr-code/index",
}
```

### API

#### Attribute 属性

| 属性 |    说明    |  类型  | 默认值 |
| ---- | :--------: | :----: | -----: |
| text | 二维码信息 | string |      - |
| size | 二维码大小 | string |      - |
| usingEncode | 开启后生成的二维码自动encodeURIComponent | string |    false |
