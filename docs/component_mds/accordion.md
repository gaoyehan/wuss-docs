# Accordion 手风琴

### 使用指南

在 page.json 中引入组件

```json
"usingComponents": {
  "w-accordion": "wuss-weapp/w-accordion/index",
}
```

### API

#### Attribute 属性

| 属性     |     说明     |  类型   | 默认值 |
| -------- | :----------: | :-----: | -----: |
| key      |  携带的数据  | string  |   null |
| title    |     标题     | string  |      - |
| disabled | 是否开启禁用 | boolean |      - |
