# Switch 开关

### 使用指南

在 page.json 中引入组件

```json
"usingComponents": {
  "w-switch": "wuss-weapp/w-switch/index",
}
```

### API

#### Attribute 属性

| 属性     |              说明               |  类型   | 默认值 |
| -------- | :-----------------------------: | :-----: | -----: |
| size     | 大小,可选[small/default/larger] | string  |      - |
| value    |                                 | boolean |  false |
| color    |       颜色，激活后的颜色        | string  |      - |
| disabled |            是否禁用             | boolean |  false |
